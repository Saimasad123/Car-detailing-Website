# Luxe Shine Detailing – Car Detailing Booking System

A full-stack car detailing service booking platform built with HTML, CSS, JavaScript, Node.js, and Express.js.

### Features

* User Registration and Login
* Email-Based OTP Verification during account registration
* Secure User Authentication
* Car Detailing Service Booking System
* Automated Booking Confirmation Emails with Complete Booking Details
* Booking History Management
* Multi-User Support
* Responsive and Modern UI/UX
* RESTful API Architecture
* JSON-Based Data Persistence

### Workflow

1. User registers an account.
2. An OTP verification code is sent to the user's Gmail address.
3. After successful verification, the user can log in.
4. Users can browse detailing services and create bookings.
5. A booking confirmation email containing service details, date, time, and customer information is automatically sent to the user's email.
6. Users can view their booking history through their dashboard.

### Technologies Used

* Frontend: HTML5, CSS3, JavaScript
* Backend: Node.js, Express.js
* Email Service: Nodemailer (Gmail SMTP)
* Data Storage: JSON File Database
* Authentication: OTP Email Verification

This project demonstrates full-stack web development, email authentication, booking management, API integration, and responsive user interface design.
