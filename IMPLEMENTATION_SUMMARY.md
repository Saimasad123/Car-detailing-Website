⚙️ CONFIGURATION OPTIONS

Default (Local AI - No setup):
  USE_AI_SERVICE=local

With OpenAI:
  USE_AI_SERVICE=openai
  OPENAI_API_KEY=sk-your-key-here

With Hugging Face:
  USE_AI_SERVICE=huggingface
  HF_TOKEN=your-token-here
